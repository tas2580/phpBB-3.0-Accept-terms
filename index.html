phpBB 3.0 Accept terms

This is an modification for the phpBB forums software. You need an instalation of phpBB 3.0.x to use this.

DESCRIPTION

Shows a list with similar topics at the end of a topic.

INSTALATION

To install this modification you need to edit some files of your phpBB installation with a text editor. Open the install.xml with your webbrowser and follow the instructions.

SUPPORT

You can get support for this extension on phpbb.com or in german on phpbb.de. For more informations look at my Website.
